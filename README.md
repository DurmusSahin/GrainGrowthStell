# Predicting Grain Growth Kinetics in Steels Using Machine Learning and XAI for Mechanical Properties

This repository contains the dataset and source codes for the study **“Predicting Grain Growth Kinetic in Steels Using Machine Learning and XAI for Mechanical Properties.”** The aim of this work is to model grain growth behavior in steels and analyze its relationship with mechanical properties using machine learning (ML) and explainable AI (XAI) techniques.

## Repository Structure
- **`Codes/`**: Python scripts and source codes for data preprocessing, model training, evaluation, and XAI analysis.  
- **`Dataset/`**: Collected experimental and/or simulated data on grain growth kinetics in steels.
